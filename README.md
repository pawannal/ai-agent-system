# 🤖 AI Agent System

A production-ready AI agent built using LangChain with tool calling and conversational memory.

---

## 🚀 Features

* ReAct-based reasoning (Thought → Action → Observation)
* Multi-tool support (Calculator, Word Counter)
* Conversational memory (context-aware responses)
* LLM-powered decision making (OpenAI)

---

## 🧠 Architecture

User → Agent → Tools → LLM → Response

---

## 🛠 Tech Stack

* Python
* LangChain
* OpenAI
* FastAPI (next step)

---

## ▶️ Run

```bash
python app/agent.py
```

---

## 📌 Example

* "What is 10 + 5?" → Uses calculator
* "My name is Pawan" → Stored in memory
* "What is my name?" → Uses memory
